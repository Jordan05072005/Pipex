# Pipex

## What is the goal of project ?
This is a 42 project whose aim is to recreate the functioning of a pipe (shell). It introduces the concept of creating a new process during a code execution (father and child process) and managing the communication channel between the two processes (pipe) with the pipe() and fork() functions.

## What functions are allowed?
-> open(), close(), read(), write(), malloc(), free(), perror(), strerror(), access(), dup(), dup2(), execve(), exit(), fork(), pipe(), unlink(), wait(), waitpid()
